# TailwindCSS Gem

> **Experimental Project** — This is an experimental project under active development. Not recommended for production use.

**Crate:** `tailwind`  
**Gem Name:** `tailwind`  
**Kind:** `Theme`

Build-time CSS generation using TailwindCSS — scans your source files for CSS classes, generates only the styles you use.

---

## Architecture

```
before_build() hook
  │
  ├── Load tailwind.config.toml
  ├── Scan content files (*.rs, *.html) for class="..." patterns
  ├── Extract unique class names
  ├── Generate CSS via tailwind-rs-core
  └── Write output CSS file
```

---

## Installation

```toml
[dependencies]
tailwind = { path = "../../products/gems/crates/tailwind" }
```

---

## Usage

### Via Boot

```rust
use viontin::boot;
use viontin_gem_tailwind::Tailwind;

fn main() {
    boot()
        .gem(Tailwind::load())
        .serve("127.0.0.1:3000");
}
```

The gem hooks into `before_build()` and generates CSS automatically during the boot sequence.

### Standalone Compilation

```rust
use viontin_gem_tailwind::compile_project;
use std::path::Path;

let css = compile_project(Path::new(".")).unwrap();
// CSS string with all used Tailwind classes
```

### Compile Raw Source

```rust
use viontin_gem_tailwind::compile_source;

let html = r#"<div class="flex items-center p-4 bg-blue-500 text-white">Hello</div>"#;
let css = compile_source(html).unwrap();
```

## Configuration

Create `tailwind.config.toml` in your project root:

```toml
[output]
path = "assets/tailwind.css"
```

### Defaults

| Field | Default |
|-------|---------|
| `output.path` | `assets/tailwind.css` |

Without a config file, output goes to `assets/tailwind.css`.

---

## Content Scanning

The gem scans these file patterns for `class="..."` and `class='...'` attributes:

| Pattern | Description |
|---------|-------------|
| `src/**/*.rs` | Rust source files |
| `src/**/*.html` | HTML templates in src |
| `html/**/*.html` | HTML templates in html/ |

Classes are extracted, deduplicated, and passed to `tailwind-rs-core` for CSS generation.

### Example

```rust
// src/pages/home.rs — detected class="..."
fn home_page() -> String {
    r#"<div class="flex flex-col items-center justify-center min-h-screen bg-gray-100">"#.into()
}
```

Generated CSS will include only the classes used: `flex`, `flex-col`, `items-center`, `justify-center`, `min-h-screen`, `bg-gray-100`.

## API Reference

### Tailwind — Gem

| Method | Input | Description |
|--------|-------|-------------|
| `load()` | — | Create the TailwindCSS gem (implements `GemBuilder` + `GemBinding`) |

### Standalone Functions

| Function | Input | Return | Description |
|----------|-------|--------|-------------|
| `compile_project(root)` | `&Path` | `Result<String>` | Compile TailwindCSS from project source files |
| `compile_source(content)` | `&str` | `Result<String>` | Compile TailwindCSS from raw HTML string |

## Complete Example

```toml
# tailwind.config.toml
[output]
path = "public/css/app.css"
```

```rust
// src/main.rs
use viontin::{boot, html};
use viontin_gem_tailwind::Tailwind;

fn main() {
    boot()
        .gem(Tailwind::load())
        .get("/", |_| {
            Response::html(html!("templates/index.html"))
        })
        .serve("127.0.0.1:3000");
}
```

```html
<!-- templates/index.html -->
<div class="min-h-screen bg-gray-50 flex items-center justify-center">
    <h1 class="text-4xl font-bold text-blue-600">Hello, Tailwind!</h1>
</div>
```

On boot:
1. Gem scans `templates/index.html` and `src/main.rs`
2. Extracts classes: `min-h-screen`, `bg-gray-50`, `flex`, `items-center`, `justify-center`, `text-4xl`, `font-bold`, `text-blue-600`
3. Generates CSS with only those utilities
4. Writes to `public/css/app.css`
